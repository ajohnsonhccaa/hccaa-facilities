Setup Guide
Three steps. ~10 minutes total. Do this once, then the form works for everyone.
How it works: The form POSTs JSON to a Power Automate HTTP trigger. The flow catches it and creates a new item in your SharePoint list. Staff never sign in to anything — they just fill out the form and hit Submit.
1 Create the SharePoint List
a
Go to your Facilities SharePoint site → click + New → List → Blank list
Name it: Equipment Inventory. Description: Master inventory of all HCCAA center equipment. Toggle "Show in site navigation" ON.
b
Rename the default 'Title' column to 'Equipment Name'
Click the Title column header → Column settings → Rename → type Equipment Name → Save.
c
Add all columns below using '+ Add column'
Click + Add column at the far right of the list header. Add each row in the table in order — the order controls how they appear in the SharePoint form.

2 Build the Power Automate Flow
a
Go to make.powerautomate.com → Create → Instant cloud flow
Name it: HCCAA Equipment Inventory — HTTP to SharePoint. For trigger, search and select "When an HTTP request is received". Click Create.
b
In the trigger step, click 'Use sample payload to generate schema'
Go to the JSON Schema tab in this form, copy the full schema, paste it into the field, and click Done. The trigger will auto-configure all the fields.
c
Click '+ New step' → search 'SharePoint' → select 'Create item'
Site Address: https://hccaainc.sharepoint.com/sites/HCCAAFacilities
List Name: Equipment Inventory
Then map each field: click the value box → select the matching dynamic content from the HTTP trigger body.
d
Save the flow → copy the HTTP POST URL from the trigger step
After saving, the trigger step shows a long URL starting with https://prod-…logic.azure.com/workflows/…. Copy the whole thing.
Field mapping tip: In the Create item step, the SharePoint column names must match exactly. If your column is "Equipment Name" the dynamic content token will be named "EquipmentName" (no space). Match them one by one down the list.
3 Connect the Form
a
Go to the Connect tab in this form
Paste the HTTP POST URL from Power Automate into the URL field → click Save URL. The status indicator at the top will turn green once the connection is verified.
b
Click 'Send Test Record' to verify the flow is working
This sends a test item to your SharePoint list. Open the list and confirm the test row appeared. Then delete it from SharePoint before distributing the form to staff.
Done. Share the saved HTML file with facilities staff via email, Teams, or a shared drive. They open it in any browser, fill it out, and click Submit — no login, no SharePoint access needed.
